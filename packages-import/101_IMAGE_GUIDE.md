## Image Specifications

- **Hero banner**: Minimum 1920 x 800px, landscape orientation
- **Highlight cards**: Minimum 600 x 400px, landscape orientation (3:2 ratio)
- **Itinerary images**: Minimum 500 x 350px, landscape orientation
- **File format**: JPEG or WebP preferred for web performance
- **File size**: Optimise to under 200KB per image for fast page loading

---

## Notes for Web Developer

1. Replace the CSS gradient placeholder divs in the HTML with actual `<img>` tags
2. Each placeholder has a class name matching its location (e.g., `.cochin`, `.munnar`) — use these as reference
3. Add `alt` text matching the descriptions in this guide for accessibility
4. Consider using `loading="lazy"` on itinerary images below the fold
5. Hero image should use `object-fit: cover` to fill the banner area
6. All images should be compressed and served in WebP format where browser support allows
